# Mining Academy Landing Page - Hostinger Ready

This is a Vite + React + Tailwind project prepared for Hostinger Node.js deployment or GitHub deployment.

## Hostinger settings

If using Hostinger Node.js/GitHub deployment:

- Install command: `npm install`
- Build command: `npm run build`
- Output directory: `dist`
- Development/start command if requested: `npm run dev`

## Upload to GitHub

1. Create a new GitHub repo named `mining-academy`.
2. Upload all files from this folder, not the ZIP itself.
3. In Hostinger, connect the GitHub repo.
4. Use the build settings above.

## Manual static upload alternative

If Hostinger lets you build the project, use GitHub deployment.
If not, you need to run `npm run build` somewhere and upload the contents of the generated `dist` folder to `public_html`.
